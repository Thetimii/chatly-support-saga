How to Embed the Chat Widget

To embed this chat widget on your website, add the following code just before the closing </body> tag:

<script 
    src="https://your-domain.com/widget.js" 
    data-website-id="YOUR_WEBSITE_ID">
</script>

Replace:

  • YOUR_WEBSITE_ID: with the website ID from your dashboard
  • https://your-domain.com: with your actual domain where the widget is hosted
